## lib Directory Overview

The **lib** folder contains all the Dart source code for the Grocery Delivery app.

### Key Files

- **main.dart** – Entry point of the Flutter application.
- **home.dart** – Home screen UI displaying categories and fruit cards.
- **model.dart** – Data models such as `Fruit`.
- **data.dart** – Sample data used to populate the UI.
- **spalsh.dart** – Splash screen shown on app launch.

### Utilities (`lib/utils`)

Utility classes like `color_app.dart` provide theming and colour constants.

### Widgets (`lib/widgets`)

Reusable UI components include:

- `badge_number.dart` – Badge for notification counts.
- `banner_widget.dart` – Promotional banner widget.
- `card_fruit_widget.dart` – Card displaying fruit information.
- `category_widget.dart` – Category selector tiles.
- `fruit_image_widget.dart` – Widget for rendering fruit images.

Each widget is self‑contained and can be extended or customized.
